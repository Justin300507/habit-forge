# Habit Forge

> Generated by **ForgeAI** — AI-powered full-stack application generator.

## Features

- CRUD operations for habits including name and target frequency.
- Ability to check off habits for the current date and view history.
- Logic to calculate current and all-time high streaks based on logs.

## Quick Start

### Prerequisites

- Python 3.11+
- Node.js 18+ (for frontend)

### Backend

```bash
cd app
pip install -r requirements.txt
uvicorn main:app --reload
```

API available at: `http://localhost:8000`
Interactive docs: `http://localhost:8000/docs`

### Frontend

```bash
npm install
npm run dev
```

Frontend at: `http://localhost:5173`

### Docker

```bash
docker build -t habit_forge .
docker run -p 8000:8000 habit_forge
```
